Les missions des femmes de ménage pour entreprise

Les femmes de ménage pour entreprise ont un cahier des charges strict à remplir, souvent défini entre l'entreprise et le client pour définir leurs services. Elles doivent notamment :

  • Respecter les horaires d'intervention dans vos bureaux.
  • Utiliser les produits de nettoyage définis en amont avec vous.
  • Dépoussiérer et nettoyer les meubles, les matériels informatiques.
  • Aérer les bureaux.
  • Nettoyer les sanitaires et les sols.
  • Vider les corbeilles, etc.

En général, la femme de ménage pour entreprise doit accomplir toutes les tâches définies avec l'entreprise de nettoyage. Cela peut inclure le lavage des vitres, le nettoyage des parties communes de l'immeuble, des ascenseurs, etc.

Si elles interviennent pendant les heures d'ouverture de vos bureaux, les femmes de ménage pour entreprise doivent travailler de manière discrète pour ne pas perturber le travail des collaborateurs.
